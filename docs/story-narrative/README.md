# 📖 Story & Narrative

Plugins that deepen the storytelling — from moral choices that shape the ending to companions that react to your journey.

| Plugin | Version | Description |
|--------|---------|-------------|
| [Morality System](morality-system.md) | 1.0.0 | Light/Dark alignment based on story choices — affects endings, NPCs, unlocks |
| [Relationship System](relationship-system.md) | 1.0.0 | Track relationship points with story companions — unlocks cutscenes, team access |
| [Flashback System](flashback-system.md) | 1.0.0 | Playable flashback sequences from other characters' perspectives |
| [Memory Lane](memory-lane.md) | 1.0.0 | End-game team journey recap with lifetime stats visualization |
| [Lore Codex](lore-codex.md) | 1.0.0 | Collectible lore encyclopedia — fragments found via books, steles, NPCs |
| [Party Banter](party-banter.md) | 1.0.0 | Context-sensitive dialogue from following Pokémon and story companions |

---

## How They Connect

```
Player makes a choice
  → Morality System shifts alignment
  → Relationship System adjusts companion points
  → Party Banter reacts with new dialogue
  → Lore Codex may unlock hidden entries
  → Memory Lane records the moment for end-game recap
```
