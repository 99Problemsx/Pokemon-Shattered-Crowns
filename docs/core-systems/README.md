# 🔧 Core Systems

Foundational plugins that the rest of Shattered Crowns builds upon. These handle data management, quality-of-life, and the level progression framework.

| Plugin | Version | Description |
|--------|---------|-------------|
| [Script System](script-system.md) | 1.0.0 | Complete PBS replacement with Ruby script definitions |
| [QoL Toggles](qol-toggles.md) | 1.0.0 | Player-facing toggle menu: auto-run, fast text, battle speed |
| [Level Caps Ex](level-caps-ex.md) | 2.3.2 | Flexible level cap system with 3 modes, badge caps, EXP storage, EV conversion |

---

## Dependencies

These plugins have **no dependencies** on other \[SC\] plugins and can be used standalone.
