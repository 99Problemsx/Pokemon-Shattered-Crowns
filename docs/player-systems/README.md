# 👤 Player Systems

Plugins that give the player new tools, progression paths, and activities beyond battling — from a branching skill tree to a photo mode.

| Plugin | Version | Description |
|--------|---------|-------------|
| [Destiny Board](destiny-board.md) | 1.0.0 | Branching skill-tree for the protagonist — spend Fate Points on passive perks |
| [Trainer Card Ex](trainer-card-ex.md) | 1.0.0 | Enhanced trainer card with achievement badges, stats, and team showcase |
| [Photo Mode](photo-mode.md) | 1.0.0 | Pause + free camera, filters, Pokémon posing, PNG export |
| [Quest Journal](quest-journal.md) | 1.0.0 | Main and side quest tracking with markers and rewards |
| [Wanted Board](wanted-board.md) | 1.0.0 | Rotating bounty board quests in cities — hunt targets for rewards |
| [Reputation System](reputation-system.md) | 1.0.0 | Faction reputation affecting shops, quests, and NPC reactions |
| [Crafting System](crafting-system.md) | 1.0.0 | Material collection and item crafting with recipe discovery |
