# 🌍 World & Exploration

Plugins that make the world of Shattered Crowns feel alive, mysterious, and rewarding to explore — from fog-covered maps to Yggdrasil-powered time travel.

| Plugin | Version | Description |
|--------|---------|-------------|
| [Map Fog of War](map-fog-of-war.md) | 1.0.0 | Town Map starts blacked out — regions reveal as you explore |
| [Time Travel](time-travel.md) | 1.0.0 | Certain locations exist in Past/Present eras — switch via Yggdrasil roots |
| [Spirit Realm](spirit-realm.md) | 1.0.0 | Alternate-dimension dungeons via Yggdrasil portals — shadow encounters, void corruption |
| [Weather Events](weather-events.md) | 1.0.0 | Gameplay-affecting weather: rare encounters, hidden areas, special NPCs |
| [Secret Base](secret-base.md) | 1.0.0 | Player-built hidden base with decorations, features, and NPC invitations |

---

## The Yggdrasil Connection

Several exploration plugins tie into the Norse mythology theme:

- **Time Travel** uses Yggdrasil roots as portals between eras
- **Spirit Realm** connects through Yggdrasil to Nidhoggr's corrupted dimension
- **Weather Events** can be triggered by mythological forces
