# 🐉 Pokémon & Training

Plugins that expand how Pokémon grow, bond with the player, and become unique through runes, titles, and legendary relics.

| Plugin | Version | Description |
|--------|---------|-------------|
| [Pokémon Affection](pokemon-affection.md) | 1.0.0 | Affection value per Pokémon granting in-battle bonuses |
| [Pokémon Titles](pokemon-titles.md) | 1.0.0 | Pokémon earn titles/epithets based on battle achievements |
| [Companion Pokémon](companion-pokemon.md) | 4.0.0 | Overworld follower system (replaces Following Pokémon EX) |
| [Rune System](rune-system.md) | 1.0.0 | Equippable Norse runes found at shrines — stat boosts socketed per Pokémon |
| [Rune Forging](rune-forging.md) | 1.0.0 | Combine two runes at Yggdrasil to create hybrid runes with combined effects |
| [Ancestral Weapons](ancestral-weapons.md) | 1.1.0 | Legendary relics tied to the Three Chosen that evolve through combat |
| [Move Tutor Network](move-tutor-network.md) | 1.0.0 | Distributed Move Tutors across both regions with tracking and discovery |
