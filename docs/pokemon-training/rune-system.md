# Rune System

**Version:** 1.0.0\
**Module:** `RuneSystem`

Equippable Norse runes discovered at shrines throughout the world. Each rune provides stat boosts when socketed onto a Pokémon — a second equipment layer alongside held items.

## Features

- Discover runes at world shrines
- Socket runes onto individual Pokémon
- Stat boost multipliers per rune
- Separate from held items (both can be equipped)
- Norse-themed naming and lore

## Configuration

See `000_Config.rb` — `ENABLED = true` to activate.
